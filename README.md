# 个人网站

林水月给朋友看的个人小站。它不是简历，而是用来分享近况、兴趣和生活碎片的互联网角落。无需安装依赖，双击即可浏览。

## 本地预览

直接双击 `index.html`，或在浏览器中打开它。

如果电脑已经安装 Python，也可以在此目录运行：

```powershell
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 继续丰富

可以继续在 `index.html` 中补充这些内容：

- 最近喜欢的歌、电影或游戏
- 想分享给朋友的照片
- 日常碎碎念和近况
- 常用社交账号（如果愿意公开）

站点颜色集中定义在 `styles.css` 顶部的 `:root` 中，可以直接调整。

## 当前功能

- 响应式桌面与手机布局
- 可记住选择的“月色模式”
- 保存在当前浏览器中的朋友留言板
- 网页图标与链接分享信息

> 留言板目前使用浏览器本地存储，不会把留言同步到其他人的设备。若要做真正的在线留言板，需要在发布后接入数据库服务。

## 打包发布

在项目目录运行：

```powershell
.\package-site.ps1
```

脚本会生成 `lin-shuiyue-site.zip`。压缩包只包含网站运行必需的四个文件，可以上传到任意静态网站托管平台。
