# 项目前端设计约定

本项目的所有前端新增、修改和重构，都必须先完整阅读并遵守：

- `.agents/skills/frontend-design/SKILL.md`

## 当前设计方向

- 主题：林水月写给朋友的“月下桌面”，不是简历或作品集。
- 单一任务：首页让朋友搜索、看时间、听音乐、与宠物互动，再继续浏览近况。
- 记忆点：沉浸式月夜首屏 + 左侧滑出的宠物音乐控制台。
- 克制原则：首屏和宠物是视觉重点，其余内容保持安静，不叠加无意义装饰。
